> 一个用react开发的appstore列表页面，使用本地mock数据  技术栈 react+redux+react-redux+react-router+axios+scss

### 截图

![](https://raw.githubusercontent.com/fozero/appstore-react/master/shortscreen/applist.png)

### 项目结构
```
|--config    
|--node_modules
|--public
|--scripts
|--src
    |-----api   //api接口
    |-----components  //组件
    |-----pages  //页面
    |-----plugins   //插件相关 axios
    |-----router  //路由
    |-----store   //redux
    |-----styles   //公共样式
    |-----utils   //工具包
    |-----index.js   //入口
|--package.json
```

### 项目构建运行
```
#运行
npm start  
#测试
npm test  
#构建
npm run build  
```
